    Class Response

    Response class for indicating the result of processing each message. Use static methods: ok(), failure(), fallback(), serve(), onSuccess()

    Example

    // Success
    sink.Response.ok(datum.id)

    // Failure
    sink.Response.failure(datum.id, 'Error message')

    // Fallback for retry
    sink.Response.fallback(datum.id)
